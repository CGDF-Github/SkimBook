# 列表查询

列表名 含义
notice 公告
node 节点
menu 节点菜单
order 订单
finance 财务
invite 邀请记录
traffic 流量记录
例：查询notice（公告）列表：
GET notice-columns
GET notice-list
GET notice-pick

# 其他

# POST menu-adjust

调整订单

参数示例：{
    "Menus":[
        {
            "HashId":"d134",
            "Operation":1,
        }
    ]
}
请求参数 说明 是否必须
Menus 菜单数组 是
Menus:HashId 菜单hash id 是
Menus:Operation 操作，1：立即激活，2：次周期激活,3：立即停用，4：次周期停用 是
