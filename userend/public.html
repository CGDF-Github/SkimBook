# 其他

GET node-menu:

获取指定节点的套餐
请求参数：
language：语言
node:node的hash id，可多个

返回：{
    "result":1,
    "menus":[
        [//第1个node的menu
            {
                "hashId":"k19323a",
                "rent":3.0,    //周期租金
                "ratio":2.0    //流量价格（？元/G）
                "speed":10    //限速，单位M
            }
            {
                "hashId":"sd545d",
                "rent":1.0,    //周期租金
                "ratio":5.0    //流量价格（？元/G）
                "speed":10    //限速，单位M
            }
        ],[//第2个node的menu
            {
                "hashId":"g4423a",
                "rent":3.0,    //周期租金
                "ratio":2.0    //流量价格（？元/G）
                "speed":10    //限速，单位M
            }
        ]
    ]
}

GET default:

获取默认配置
请求参数：
language：语言

返回：{
    "result":1,
    "languages":"en",    //默认语言    
    "timezone":"Asia/Shanghai",    //本站使用的时区
    //https://en.wikipedia.org/wiki/List_of_tz_database_time_zones
    //数据库总是以设定的时区存储
    //除非特别说明，API默认返回设定的时区    
    "currency":"CNY",    //最终结算货币
    //https://en.wikipedia.org/wiki/ISO_4217
    "cycle":7    //本站账单周期
    ]
}

GET resource：

获取资源（比如背景图什么的）
请求参数：
name=background-public（资源名）

返回：{
    "result":1,
    "resource":"..."
}
