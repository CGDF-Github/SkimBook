### 主要设计

| 设计 | 说明 |
| :--- | :--- |
| 国际化 | 支持多个\[语言、货币、时区\] |
| 计费模式 | 使用`固定租金+流量计费`模式 |
| 用户角色 | 细化用户权限，支持使用不同的权限组合为`角色`，再将`角色`分配至不同的成员 |
| 设置项 | 设置项全部保存在数据库中，并且可以通过\[网站后台/客户端\]修改 |
| 简易负载均衡 | 支持简易负载均衡，可以通过多台服务器上分摊流量 |
| 安全性 | 强制使用https和wss |
|  | \[客户端、后端、节点端\]之间使用websocket双向通信，实时更新数据 |
